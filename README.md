## 背景
这个软件目前只是提供门厂的销售人员使用。软件的用途是将不同款式的门框和不同款式的门自由搭配合成，展示不同效果给用户自由选购

## 硬件支持
软件支持在安卓手机，安卓平板电脑上运行
